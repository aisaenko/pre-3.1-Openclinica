5.1 Administer Users Help

This section allows you to view all the users available to you. All users are ordered through their User Name. You can choose to reorder the list of users by toggling the ascending or descending arrow located next to the User Name header. Similarly, you can also reorder the subject list using the user’s First Name, Last Name, or Status, and toggle their respective ascending or descending arrows.

Note: Under each First and Last Name are listed the user’s role within specific studies given in a “Study Name – Study Role” format.

1. Find

The Find function, located at the top of the user list, allows you to search for a set of users based on their User Name, First Name, or Last Name.

2. First Name

This cell indicates the first name of each user.

3. Last Name

This cell indicates the last name of each user.

4. Status

This cell indicates the status of each user.

5. Actions View

Clicking on the View button will allow you to see all the details associated with the particular user.

Edit

Clicking on the Edit button will allow you to edit the user’s First Name, Last Name, Email, Institutional Affiliation, and User Type.

Set Role

Clicking on the Set Role button will allow you to assign a Role within a Study for the particular user.

Remove

Clicking on the Remove button will allow you to remove the particular user. From this point on, the user will no longer be associated with any study and any roles, unless the user is restored.

Restore

Clicking on the Restore button will allow you to restore the particular user. From this point on, the user will have access to the assigned studies under specific roles, unless the user is removed.

In addition to viewing all the Users, you are also presented with a link to Create a new User.
