3.3 View Filters Help

This section allows you to view all the filters that have been previously created. All filters are ordered using their Filter Name. You can choose to reorder the list of filters by toggling the ascending or descending arrow located next to the Filter Name header. Similarly, you can also reorder the subject list using the dataset’s Description, Created By, Created Date, or Status, and toggle their respective ascending or descending arrows.

1. Find

The Find function, located at the top of the filter list, allows you to search for a set of filters based on their Filter Name or Description.
2. Description

This cell provides a brief description for each filter.
3. Created By

This cell indicates the creator of each filter.
4. Created Date

This cell indicates the date the filter was created.
5. Status

This cell indicates the current status of each filter.
6. Actions View
Clicking on the View button will allow you to see all the details associated with the particular filter.
Edit
Clicking on the Edit button will allow you to edit the Name, Description, and Status of the particular filter.
Remove
Clicking on the Remove button will allow you to remove the particular filter. From this point on, only the Administrator can access and reverse this change if necessary.
