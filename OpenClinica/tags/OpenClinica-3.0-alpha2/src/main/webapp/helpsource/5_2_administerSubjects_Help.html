5.2 Administer Subjects Help

This section allows you to view all the subjects that are available to you across all studies. All subjects are initially ordered by their Person ID (formerly “Global Unique ID”). You can order the list of subjects by clicking on the link at the top of each column. You can switch between ascending and descending order by clicking the arrow located next to each column title.

1. Find

The Find function, located at the top of the user list, allows you to search for a subject based on any portion of the Person ID or Protocol-SubjectID.

2. Person ID

This cell displays the Person ID, which can be used to link subjects from different studies to a single individual.

3. Protocol-SubjectIDs

This cell displays the study (short protocol ID) and study subject ID together. It is not available for sorting.

4. Sex

This cell indicates the sex of each subject.

5. Date Created

This cell indicates the date the subject was created.

6. Owner

This cell indicates the owner who originally created the subject.

7. Date Updated

This cell indicates the date the subject was last updated.

8. Last Updated By

This cell indicates the user who last updated the subject.

9. Status

This cell indicates the status of each subject.

10. Actions View

Clicking on the View button will allow you to see all the details associated with the particular subject.

Edit

Clicking on the Edit button will allow you to edit the subject’s Person ID, Sex, Date of Birth, Father, and Mother.

Remove

Clicking on the Remove button will allow you to remove the particular subject. From this point on, only the Administrator can access and reverse this change if necessary.

Restore

Clicking on the Restore button will allow you to restore the particular subject. From this point on, the subject will be available for studies, unless the subject is removed.
