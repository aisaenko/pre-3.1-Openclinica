4.7 View Study Subject Group Classes Help

This section allows you to explore all the Subject Group Classes that you have made to categorize groups of Subjects within a particular Study. You can choose to view the groups in any order, and can also create a Subject Group Class by clicking on the 'Create' link at the top of the table.

A Subject Group Class is a class of group categories that are related and mutually exclusive. For instance, when creating a group class you may create a Group Class called 'Trial Arm' with the groups 'Treatment A', 'Treatment B', and 'Placebo'. You may then assign the subject to one of these groups. If you specify the Subject Assignment as 'Required', you must assign the subject to one of these groups upon creating the subject record. You can have multiple Group Classes within a study.

You should enter a Group Class name, a type, whether enrollment in the group is required or not, and the name and description of the Subject Group(s) in the Class. Note that you can create more than one group at this screen.
