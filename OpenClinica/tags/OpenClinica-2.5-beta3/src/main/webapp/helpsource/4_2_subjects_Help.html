4.2 Subjects Help

This section allows you to view all the subjects that are a part of the current study. All subjects are ordered through their Study Subject Identifier. You can order the list of subjects by clicking on the link at the top of each column. You can switch between ascending and descending order by clicking the arrow located next to each column title.

1. Find

The Find function, located at the top of the subject list, allows you to search for a set of users based on their Unique Identifier, Gender, or Enrollment Date.

2. Gender

This cell indicates the gender of each subject.

3. Enrollment Date

This cell indicates the enrollment date of each subject.

4. Study Events

For each subject, the number of Study Events is listed along with the number of Study Events that have not been started in parenthesis.

5. Status

This cell indicates the status of each subject.

6. Actions View

Clicking on the View button will allow you to see all the details associated with the particular subject.

Remove

Clicking on the Remove button will allow you to remove the particular subject from the study. From this point on, only the Administrator can access and reverse this change if necessary.

Reassign

Clicking on the Reassign button will allow you to reassign the particular subject to another site within the study. From this point on, the subject will be associated with the new site.

Restore

Clicking on the Restore button will allow you to restore a removed subject. From this point on, the subject will be assigned back to the specific study, unless the user is removed.

In addition to viewing all the subjects, you are also presented with a link to Add a New Subject.
